# MIG (Minimal Image Generator)

This library is intended to be used for drawing basic shapes onto an image in memory which can then be saved as an image file.

#### Supported shapes

* Circles
* Lines
* Pixels or Dots
* Rectangles

#### Supported image output types

* PNG
* BMP
* PPM (ASCII)
